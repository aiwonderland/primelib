A library related to prime numbers, which provides a wealth of functions about prime numbers.

```python
>>> import primelib as pb
>>> pb.is_prime(12)
False
>>> pb.is_prime(1000000007)
True
>>> pb.not_prime(12344)
True
```

This class has a great many functions; for instance, it outputs numerous prime numbers, and it is an iterator.
